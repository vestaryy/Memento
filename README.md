Это веб-сайт, который помогает пользователю сохранять важные моменты жизни. Вы загружаете фотографию и описываете свои чувства и мысли в этот момент. Раз в месяц искусственный интеллект анализирует все ваши записи и присылает краткий отчет с самыми яркими событиями.
Как это работает технически:
 1. Сайт написан на языке Python с использованием фреймворка Flask.
 2. Для хранения данных используется база данных SQLite и библиотека SQLAlchemy. В ней хранятся данные пользователей и тексты их воспоминаний.
 3. Сами фотографии не занимают место на сервере сайта. Они сразу пересылаются в облако Яндекс Диска через API. Сайт хранит только пути к этим файлам.
 4. При просмотре страницы сайт запрашивает временные ссылки у Яндекса, чтобы показать вам ваши фото.
 5. Для безопасности используется система регистрации и входа, а пароли хранятся в зашифрованном виде.

 Структура папок:
 * data: здесь лежат настройки базы данных и описание таблиц.
 * forms: здесь лежат формы для регистрации, входа и загрузки фото.
 * templates: здесь хранятся HTML-страницы сайта.
 * utils: здесь находится код для работы с Яндекс Диском.
 * db: папка, в которой создается файл самой базы данных.
