Проект представляет собой персонального ассистента для накопления и анализа личной информации в течение месяца. Бот реализован на языке Python с использованием библиотеки aiogram для взаимодействия с интерфейсом мессенджера. Основная задача инструмента заключается в сборе разнородного контента, такого как текстовые заметки, фотографии, видеосообщения и ссылки, с их последующей систематизацией.
Для управления зависимостями проекта применяется файл requirements.txt, в котором зафиксированы версии всех необходимых библиотек. Хранение данных организовано в базе данных, управление которой осуществляется через SQLAlchemy. Использование этой ORM-системы позволяет абстрагировать взаимодействие с базой данных, представляя таблицы в виде Python-классов. Это обеспечивает надежную типизацию, упрощает построение сложных запросов для ежемесячной фильтрации контента и позволяет легко переключаться между различными СУБД без изменения основного кода приложения. В базе фиксируются идентификатор пользователя, тип контента, путь к загруженному файлу, текст сообщения и временная метка.
В логике приложения реализована работа с контекстом пользователя через машину состояний. Это позволяет боту корректно обрабатывать последовательные действия, например, подвязывать описание к только что загруженной фотографии или распределять контент по категориям в режиме диалога. Бот поддерживает загрузку и хранение медиафайлов: изображения и аудиофайлы сохраняются на сервере, а ссылки на них записываются в базу данных для последующего воспроизведения или анализа.
Интеллектуальная часть проекта реализована через интеграцию со сторонним API нейросетевых моделей. Раз в месяц система извлекает все накопленные через SQLAlchemy данные за отчетный период и отправляет их в обработку. Внешнее API анализирует текстовые сообщения и описания к файлам, формируя краткую сводку и выделяя ключевые события месяца. Готовый отчет отправляется пользователю в виде структурированного сообщения. Приложение подготовлено к развертыванию на хостинге с использованием фоновых задач для автоматической генерации ежемесячных отчетов.
