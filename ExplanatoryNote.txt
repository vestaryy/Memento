Проект представляет собой веб-приложение для ведения личного дневника воспоминаний. Главная идея — не просто хранить фотографии, а привязывать к ним эмоциональное состояние человека.
Пользователь создает записи, состоящие из медиафайла и текстового описания. В конце месяца система помогает пользователю отрефлексировать прошедший период, собирая все данные в единый ИИ-отчет.
Цели проекта:
 1. Создать удобный инструмент для хранения личных архивов.
 2. Реализовать безопасную передачу файлов в облако (Яндекс Диск) без захламления памяти сервера.
 3. Внедрить элементы искусственного интеллекта для анализа текстовых заметок пользователя.
Используемые технологии:
 * Flask: основной веб-движок приложения.
 * SQLAlchemy: система для работы с базой данных через объекты Python. Она позволяет легко управлять пользователями и их записями.
 * Яндекс Диск API: используется для хранения всех изображений. Реализован алгоритм, который передает байты файла
  напрямую с компьютера пользователя в облако через сервер приложения.
 * Flask-Login: обеспечивает работу личных кабинетов. Каждый пользователь видит только свои записи.
 * WTForms: технология для создания защищенных форм ввода данных (регистрация, загрузка контента).
 * SQLite: легкая база данных для хранения информации о пользователях и путях к их файлам.
